## 📌 Featured Projects

### 🚤 Boat Reservation API

Backend REST API for managing sailors, boats and reservations.

**Main features:**
- Create, list, update and delete sailors
- Manage boats
- Create and cancel future reservations
- Connect backend logic with an Oracle database

**Technologies:** Node.js, Express.js, Oracle Database, Postman

Repository: [web-api-barcos](https://github.com/catarinasoares94/web-api-barcos)

Frontend application built with Vue.js to interact with the boat reservation API.

**Main features:**

- Manage boats
- Create and view reservations
- Form validation and API response handling

**Technologies:** Vue.js, JavaScript, HTML5, CSS3, Vite

Repository: [web-api-barcos-frontend](https://github.com/catarinasoares94/web-api-barcos-frontend)
